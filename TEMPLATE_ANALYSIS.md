# Portfolio Template Analysis

## Template Structure (from PDF)

### Pages:
1. **Home**: Hero section with greeting, name, description, social icons, "Get Started" button, profile image on right
2. **About**: Profile image on left, "About Me" heading, description, "Resume" button, social icons
3. **Projects**: Filter tabs (All, Space Science, SWE, Electronics, 3D Modeling, Design/Art), project cards in grid with image, title, description, tech tags, "More" button
4. **Experience**: Tabs for Experience, Download Resume, Skills, Education; skill bars; education cards; honors/awards grid
5. **Contact**: Contact form (Email, Subject, Message), phone/email info, social icons, yellow wave divider

### Design Elements:
- **Color Scheme**: Yellow (#FFFF00), Black, White, Light Gray background
- **Typography**: Bold serif for headings, sans-serif for body
- **Layout**: Horizontal navigation, asymmetric layouts, image-heavy
- **Social Icons**: GitHub, LinkedIn, Email, Telegram (circular with border)
